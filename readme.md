# MegaStore E-Commerce Website

MegaStore is a responsive frontend e-commerce project built with HTML, CSS, and JavaScript. It allows users to browse products, search items, add products to the wishlist or cart, register or log in, and explore different categories.

## ✨ Features
- Home page with a slider, category cards, featured products, and offer banners
- Product page with category-based filtering and search
- Product detail page for viewing product information
- Wishlist and cart management
- Order summary and coupon section on the cart page
- Login and registration flow
- About page with team details
- Theme toggle for dark/light mode
- Newsletter subscription section

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage for saving user, cart, and wishlist data

## 📁 Project Structure
- `index.html` — Home page
- `pages/` — All main website pages such as products, wishlist, cart, login, and about
- `css/` — Styling files for the full website
- `js/` — JavaScript logic for app behavior and interactivity
- `images/` — Images and icons used across the project

- No backend is connected, so all data is handled on the client side.
